# TestQual

## Test if type `T1` is convertible to `T2`. Both types `T1` and `T2` are made of only tokens `char`, `const`, `*`, `[]`

### How to run executable

#### After building the project one can run an executable:
```
./build/Lecture-1/testqual/testqual "<T1>" "<T2>"
```

### How to run tests

#### There is a python script which generates a lot of tests and runs them all. One can run the tests:
```
cd test
python test.py
```
